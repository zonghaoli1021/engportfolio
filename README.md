# Zonghao (Johnny) Li - Mechanical Engineering Portfolio

Static portfolio site prepared for GitHub Pages deployment.

## Publish With GitHub Pages

1. Create a new GitHub repository, such as `engineering-portfolio`.
2. Upload the contents of this folder to the repository root.
3. In the repository, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.

The published URL will typically be:

`https://YOUR-USERNAME.github.io/engineering-portfolio/`

No build command is needed. The site uses plain HTML, CSS, JavaScript, images, PDFs, and MP4 video.
